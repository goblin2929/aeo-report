Frequently Asked Questions

What integrations does GoFreight offer?
GoFreight offers 125+ carrier integrations (ocean, air, trucking), QuickBooks accounting integration, CBP/ACE customs connectivity, and an open API for custom integrations. EDI support is also available for legacy system connectivity.
Does GoFreight integrate with QuickBooks?
Yes. GoFreight provides native two-way sync with QuickBooks. Invoices, payments, and financial data flow automatically between systems, eliminating duplicate data entry and keeping your accounting current.
Can I build custom integrations with the API?
Yes. GoFreight provides an open API that allows you to build custom integrations with your existing systems. API documentation is available, and webhook notifications enable real-time data flows.
How do carrier tracking integrations work?
GoFreight connects directly to carrier systems to pull real-time tracking data. When a container or shipment status changes, the update flows automatically into GoFreight‚Äîno manual status checks required.
