Component Gap Audit

Air Freight Solutions Page — Missing Components vs. Live

Comparison between our proposed draft and the version GoFreight shipped. The live page kept the core narrative (problem section, 4 solution blocks) and added customer logos + product screenshots, but dropped several components we proposed that materially affect SEO/AEO performance and quantified credibility. Every missing block below is delivered as drop-in HTML matching the original design tokens so it can be lifted straight onto the live page.

Proposed draft: air-v2.html
Live page: gofreight.com/solutions/air-freight
Date: 2026-05-11

1. Summary — What's Missing on the Live Page

Five proposed components were dropped on the shipped version. The live page also added two components we did not propose (customer logo strip + per-block product screenshots), which are net positives. Below: every component, status, and the SEO/AEO/UX impact of leaving it off.

Note: structured data (FAQPage JSON-LD) will be added manually via in-house tools once the visible components ship, so it is not listed as a deliverable here.
